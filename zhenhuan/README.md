# 甄嬛传后宫

以后宫赛马体系激发 Agent 潜力 — 竞争驱动进化，制度约束行为，资源激励表现。

## 为什么设计这个场景

后宫是一个天然的 competitive 多 agent 系统：所有 agent 争夺同一个有限资源（圣宠），但不能直接消灭对手，必须通过联盟、策略和制度博弈间接竞争。这让关系演化非常丰富——今天的盟友可能明天叛变（安陵容），看似弱势的角色可能暗中积蓄力量（端妃）。

## 关系网络亮点

- **三角背叛**：甄嬛-安陵容-皇后 构成经典的"盟友→叛变→投靠敌方"路径
- **暗盟结构**：端妃与甄嬛的低调联盟在事件驱动下逐渐显现
- **华妃悖论**：rank 高但缺乏联盟，展示孤立节点在集群分析中的表现
